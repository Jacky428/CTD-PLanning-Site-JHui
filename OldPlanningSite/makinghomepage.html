How my planning site was created

I have created my planning site storyboard with two different applications which are Firework and Dreamweaver. I am going to discuss the strength and weakness of using Fireworks and Dreamweaver.

What is Firework? What is Dreamweaver?

Adobe Dreamweaver and Fireworks software are both commonly used by web-designers and developers around the world. However, each program is only available to a standardize level; both of them have different strength and benefit but they do contain problems and weaknesses for the users. Both of the programs are available for the Microsoft Windows and Mac OS x operating system.

Adobe Dreamweaver is a web development tool used to build websites and applications with mostly programming code, such as the most common HTML and CSS. Dreamweaver also provides a special feature that what you see is what you get; once you type out your code, you will automatically see the design and layout of your webpage. Also, Dreamweaver contains features such as the ability to manage uploading of site files in one application, to create the structure of the layout right after you create your code, and the integration with CMS (content management system, a system used to mange the content of a web site) such as WordPress.

Adobe Firework is a graphics application intended for a very broad of usage, from entertainment such as designing poster to actual business such as designing name card. We could use Firework to optimize images, create digital images, and webpages too. Firework shares similar functions with Photoshop such as the editing of pictures and images. Firework allows the function of interactive Web prototype, by exporting the webpage you created as a HTML, CSS or image; you can send the prototype to client for reference and approval.

The strength of using Firework

  • Firework allows you to create a brief working draft.
  • Firework allows you to construct layouts for publications, organize large number of text blocks and images.
  • Firework allows you to create images with quick handy vector tools.
  • Fireworks provide design with pixel perfect accuracy, Fireworks allow you to control the exact number of pixels and axis coordination you prefer in your design.
  • Fireworks allow you to build a full-working functional website. You can create buttons, navigation bars, hotspots and rollover, from the program. You can use slice tools to customized your webpage.
  • Fireworks allow you to design quick gradients, it is much easier than creating gradients in Photoshop or Dreamweaver.

    • The weakness of using Firework

  • Fireworks would experience slow performance when counters with large file image size.
  • Fireworks does not allow immediate view of your designed webpage.
  • Fireworks have limited filters, layer styles and effects.
  • Fireworks require practice to gain perfection in vector tools
